## Some useful info for you

- This is just a simple html page.
- All designs and themes are based on own telegram account. (This code has no control over it)
- In oder to work this code, you must first create your own instant view template in telegram's instant view generator.


## Key code tags

`instant` is a wrapper for which part should be converted into Iv.

`footer` is a standard html which is used for foot element, but in this case it is used for hide the said element in IV.


## From the author

Every image files are actually in webp format, But tricked into png format. Because telegram only allowed jpg or png.
